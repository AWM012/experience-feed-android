# 课后作业要求摘录

根据课程 PDF 中「课后作业」部分，作业要求整理如下：

## 完整功能实现

- 网络模块：使用 OkHttp 发起网络图片请求，可选择任意网络图片。
- 界面展示：在 Activity 中显示双列图片列表。
- 图片预览：点击图片可预览大图。
- 图片描述：图片下方显示图片描述，包括但不限于图片日期、大小、存储位置。
- 历史记录：若下次启动页面，图片已经下载过，应使用缓存，不能重复下载。
- 自由发挥：可自由补充增强功能。

## 交付物

- GitHub 仓库：包含项目完整代码和项目介绍说明。
- 演示视频：3 分钟内展示 App 功能、代码结构、技术难点和解决方案。
- 学习总结：分析网络请求过程中遇到的 3 个问题及优化思路。
